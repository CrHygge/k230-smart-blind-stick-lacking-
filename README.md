# K230 智能盲杖：基于 YOLOv8n 的避障探索项目

本项目为基于嘉楠 K230 芯片的智能盲杖机器视觉避障方案，旨在探索边缘端深度学习模型部署的完整流程，包括模型训练、量化与硬件适配。

---

## 📌 项目背景
本项目为个人学习探索项目，旨在通过实践掌握 K230 平台上的 YOLO 模型部署流程。
- 硬件平台：嘉楠 K230 开发板（CanMV 环境）
- 模型方案：YOLOv8n 目标检测模型
- 应用目标：为视障人士辅助设备开发提供可复现的边缘端部署参考

---

## ⚠️ 项目说明
- **数据集**：使用公开网络数据集，因版权限制不公开。
- **模型表现**：训练模型平均置信度约 0.61，受限于数据集与任务方向，目标检测效果不佳，量化后精度下降明显。
- **经验总结**：针对盲道识别任务，目标检测方案并非最优解，后续应优先考虑语义分割模型（如 SegFormer）以提升效果。

---

## 🛠️ 环境配置
- 开发板：嘉楠 K230（CanMV IDE）
- 依赖：`CanMV 固件 v2.x`、`OpenCV`
- 模型部署：YOLOv8n 模型量化为 `.kmodel` 格式
